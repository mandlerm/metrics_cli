## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

You must have `python3` installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mandlerm/metrics_cli.git
   ```
2. In the terminal:
   ```sh
   python3 metrics_cli.py <<filename>>
   ```
